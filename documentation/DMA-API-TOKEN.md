## Usage

This token is used for accessing the Harvard Dataverse API to fetch DMA (Designated Marketing Area) data.

Set it in your `.env.local` file:
```bash
DATAVERSE_API_TOKEN=a7a73014-0cce-4ab0-b825-c629c6dd7be8
```

Or export it in your terminal:
```bash
export DATAVERSE_API_TOKEN='a7a73014-0cce-4ab0-b825-c629c6dd7be8'
```

## ⚠️ IMPORTANT REMINDER

**REGENERATE TOKEN BY: October 2025**

The token expires on **2026-11-30**. Please regenerate a new token approximately 11 months from now (around October 2025) to ensure uninterrupted service.

### How to Regenerate

1. Go to https://dataverse.harvard.edu/
2. Sign in to your account
3. Click on your profile (top right)
4. Go to "API Token" section
5. Click "Create Token" or "Regenerate Token"
6. Update this file with the new token and expiration date
